# 05 — 서재 탭 (책 목록 + 필터)

## 개요

탭 4의 메인 화면. 내가 추가한 모든 책을 상태별 탭 필터로 나눠서 볼 수 있다.
그리드 또는 리스트 형태로 책 카드를 보여주고, 카드 탭 시 `06-library-detail`로 이동한다.

**의존**: `00-swiftdata`

---

## 사용자 시나리오

1. 서재 탭 진입
2. 기본 "전체" 탭 — 모든 책 표시
3. "읽고 있는" / "완독한" / "읽고 싶은" 필터 탭으로 전환
4. 책 카드 탭 → 책 상세/히스토리 화면으로 이동

---

## 기능 명세

### 탭 필터
| 탭 | 표시 조건 |
|---|---|
| 전체 | 모든 `LibraryBook` |
| 읽고 있는 | `status == .reading` |
| 완독한 | `status == .completed` |
| 읽고 싶은 | `status == .wantToRead` |

### 책 카드 표시 정보
- 책 표지
- 제목 (2줄 말줄임)
- 저자

### 정렬
- 전체 탭: `addedAt` 내림차순 (최근 추가순)
- 읽고 있는: 최근 세션 시간 내림차순
- 완독한: `completedAt` 내림차순
- 읽고 싶은: `addedAt` 내림차순

### Empty State
- 각 탭별 빈 상태 안내 문구
- 전체 탭 비어있을 경우: [책 검색하기] CTA

### 레이아웃
- 그리드 (기본, 2열)
- 추후 리스트 전환 버튼 추가 가능 (현재 스코프 외)
