# 06 — 서재 책 상세 + 세션 히스토리

## 개요

서재 탭에서 책 카드를 탭했을 때 나오는 상세 화면.
해당 책에 대한 누적 독서 정보와 세션 히스토리를 보여주고, 상태 변경(완독 처리 등)을 할 수 있다.

**의존**: `00-swiftdata`, `05-timer-session`

---

## 사용자 시나리오

1. 서재 탭에서 책 카드 탭
2. 책 상세 화면 진입
3. 책 메타 정보 + 독서 통계 확인
4. 세션 히스토리 리스트 확인 (날짜, 독서 시간)
5. [완독으로 표시] 또는 [삭제] 등 상태 액션

---

## 기능 명세

### 표시 정보

**책 정보 섹션**
- 책 표지 / 제목 / 저자 / 출판사

**독서 통계 섹션**
- 총 독서 시간 (전체 세션 합계)
- 총 독서 횟수 (세션 수)
- 서재 추가일 / 독서 시작일 / 완독일(있을 경우)

**세션 히스토리 섹션**
- 날짜별 세션 리스트
- 각 세션: 날짜, 시작 시간, 독서 시간

### 상태 액션
- `reading` 상태: [완독으로 표시] 버튼
- `wantToRead` 상태: [읽기 시작] 버튼 (→ `reading` 전환)
- 서재에서 삭제: 스와이프 or 더보기 메뉴

## 데이터

- 입력: `LibraryBook.id`
- `LibraryRepositoryProtocol.fetchSessions(for:)` — 세션 목록
- `LibraryRepositoryProtocol.updateStatus(id:status:)` — 상태 변경
- `LibraryRepositoryProtocol.delete(id:)` — 서재에서 삭제

### 1. 화면 기본 구성

- [ ] `LibraryBook` 또는 `UUID`를 받는 이니셜라이저
- [ ] 책 메타 정보 헤더 (표지, 제목, 저자, 출판사)

### 2. 독서 통계 섹션
- [ ] 총 독서 시간 계산 및 포맷 표시
- [ ] 총 독서 횟수 표시
- [ ] 날짜 정보 (추가일, 시작일, 완독일)

### 3. 세션 히스토리 리스트
- [ ] `ReadingSession` 목록 fetch
- [ ] 날짜 포맷 (yyyy.MM.dd)
- [ ] 시간 포맷 (X시간 Y분 / Y분)
- [ ] 최신 세션이 상단에 오도록 정렬

### 4. 상태 액션 버튼
- [ ] 현재 `ReadingStatus`에 따라 버튼 조건부 표시
- [ ] [완독으로 표시] → `updateStatus(.completed)` + `completedAt = Date()`
- [ ] [읽기 시작] → `updateStatus(.reading)` + `startedAt = Date()`

### 5. 삭제 기능
- [ ] 네비게이션 바 더보기 메뉴 또는 스와이프 동작
- [ ] 삭제 확인 Alert ("서재에서 삭제하면 모든 독서 기록도 삭제돼요")
- [ ] `delete(id:)` 호출 후 이전 화면으로 pop

## 완료 기준

- [ ] 책 메타 정보와 독서 통계가 정확하게 표시됨
- [ ] 세션 히스토리가 최신순으로 표시됨
- [ ] [완독으로 표시] 탭 시 상태가 변경되고 화면에 반영됨

- [ ] 세션이 없는 책은 히스토리 섹션에 빈 상태 메시지 표시
