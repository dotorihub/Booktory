# 03 — 독서 탭 (읽고 있는 책 목록)

## 개요

탭 1의 메인 화면. `reading` 상태인 책들을 카드 리스트로 보여주고, 각 책에서 타이머를 시작하는 진입점이다.
읽고 있는 책이 없을 때는 Empty State를 표시한다.

**의존**: `00-swiftdata`, `02-library-action`

---

## 사용자 시나리오

### 시나리오 A — 책이 있는 경우
1. 독서 탭 진입
2. 읽고 있는 책 카드 목록 확인
3. 원하는 책의 [이어 읽기] 탭
4. 해당 책의 타이머 화면(fullScreenCover) 오픈

### 시나리오 B — Empty State
1. 독서 탭 진입
2. 읽고 있는 책 없음 → Empty State 화면 표시
3. [책 검색하기] 탭 → 탭 2(검색)로 전환

---

## 기능 명세

### 책 카드
- 책 표지 썸네일
- 제목 (1줄, 말줄임)
- 저자
- 누적 독서 시간 (해당 책의 전체 세션 합계)
- [이어 읽기] 버튼

### 목록 정렬
- 기본: 최근 독서 세션 시간 기준 내림차순 (가장 최근에 읽은 책이 위)
- 세션이 없으면 `startedAt` 기준

### Empty State
- 일러스트 또는 아이콘
- 안내 문구: "읽고 있는 책이 없어요"
- [책 검색하기] CTA 버튼 → 탭 2로 전환

## 데이터

- `LibraryRepositoryProtocol.fetchBy(status: .reading)` — 읽고 있는 책 목록
- `LibraryRepositoryProtocol.fetchSessions(for:)` — 누적 독서 시간 계산
- `@Query`로 실시간 반영 (책 추가/삭제 즉시 반영)
