# 03 — 책 상세 화면

## 개요

검색 결과 리스트에서 책 아이템을 탭했을 때 나타나는 상세 화면.
책의 메타 정보를 보여주고, 서재에 추가하는 액션 두 가지([바로 읽기], [읽고 싶은 책])의 진입점이 된다.

**의존**: `00-swiftdata`, `01-library-action`

---

## 사용자 시나리오

1. 검색 탭에서 책 검색
2. 결과 리스트에서 원하는 책 탭
3. 책 상세 화면으로 이동 (push navigation)
4. 책 표지, 제목, 저자, 출판사, 설명 확인
5. [바로 읽기] 또는 [읽고 싶은 책에 저장] 탭

---

## 기능 명세

### 표시 정보
- 책 표지 이미지 (coverURL, 로딩/에러 처리 포함)
- 제목
- 저자
- 출판사
- 책 소개 (description, 길 경우 "더 보기" 토글)

### 서재 상태 표시
- 이미 서재에 있는 책이면 현재 상태 표시
  - 읽고 싶은: [읽고 싶은 책 저장됨] (비활성 스타일)
  - 읽고 있는: [읽고 있는 중]
  - 완독: [완독한 책]
- 상태에 따라 버튼 레이블/스타일 변경

### 액션 버튼
- **[바로 읽기]**: 항상 표시 (읽고 있는 상태가 아닐 때만 활성)
- **[읽고 싶은 책에 저장]**: 저장되지 않은 경우에만 활성 표시
  - 이미 저장된 경우 [저장됨] 으로 표시 (탭 시 제거 토글)

## UI 명세

```
┌─────────────────────────────┐
│  < 뒤로가기                  │  ← NavigationBar
├─────────────────────────────┤
│                             │
│      [책 표지 이미지]         │  ← AsyncImage, 상단 중앙
│                             │
│  제목                        │  ← 굵게, 멀티라인
│  저자 · 출판사               │  ← 서브텍스트
│                             │
│  ─────────────────────────  │
│  책 소개                     │  ← 최대 4줄, 더 보기 토글
│                             │
├─────────────────────────────┤
│  [읽고 싶은 책에 저장]        │  ← 보조 버튼 (outline)
│  [바로 읽기]                 │  ← 주요 버튼 (filled)
└─────────────────────────────┘
```

- 액션 버튼 탭 시 `02-library-action` 로직 호출
