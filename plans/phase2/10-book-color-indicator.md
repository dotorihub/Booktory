# 10 — 책 컬러 인디케이터 + 달력 도트 표시

## 개요

기록 탭의 달력과 하단 목록을 개선하여, 각 책에 고유 컬러(빨주노초파)를 부여하고
달력 날짜 셀에 해당 날 읽은 책의 컬러 도트를 표시한다.
달력 하단에는 세션 시간 리스트 대신 **책 표지 카드** 목록을 보여준다.

**의존**: `07-stats`, `08-calendar-weekly`, `09-calendar-monthly`

---

## 사용자 시나리오

1. 서재에 책을 추가하면 자동으로 컬러가 배정됨 (빨 → 주 → 노 → 초 → 파 순환)
2. 기록 탭 진입 → 달력의 각 날짜 아래에 그날 읽은 책들의 컬러 도트 표시
3. 날짜 탭 → 하단에 해당 날 읽은 책 표지 카드 목록 표시
4. 카드에는 컬러 뱃지, 책 표지, 책 이름, 그날 읽은 총 시간이 포함됨

---

## 기능 명세

### 1. 책 컬러 시스템

#### 컬러 팔레트 (v1: 5색)

| 순서 | 이름 | 역할 |
|------|------|------|
| 0 | 빨강 (Red) | 첫 번째 책 |
| 1 | 주황 (Orange) | 두 번째 책 |
| 2 | 노랑 (Yellow) | 세 번째 책 |
| 3 | 초록 (Green) | 네 번째 책 |
| 4 | 파랑 (Blue) | 다섯 번째 책 |

#### 배정 규칙
- 서재에 책이 추가될 때 `colorIndex`를 자동 배정 (순서대로 순환: 0→1→2→3→4→0→…)
- `LibraryBook` 모델에 `colorIndex: Int` 프로퍼티 추가 (기본값 0)
- 기존 책에는 마이그레이션 시 추가 순서(`addedAt`) 기준으로 배정
- 향후 사용자가 직접 컬러를 변경하는 기능으로 확장 가능

#### 컬러 enum

```swift
enum BookColor: Int, CaseIterable {
    case red = 0
    case orange = 1
    case yellow = 2
    case green = 3
    case blue = 4

    var color: Color { ... }    // SwiftUI Color 반환
    var label: String { ... }   // 접근성 레이블 ("빨강", "주황", ...)
}
```

- 5색으로 시작, 추후 케이스 추가로 확장 용이한 구조

### 2. 달력 날짜 도트

#### 표시 규칙
- 각 날짜 셀 하단에 해당 날 읽은 책들의 컬러 도트를 HStack으로 표시
- 도트 크기: 6pt 원형
- 최대 표시 개수: 5개 (초과 시 5개까지만 표시)
- 같은 책의 세션이 여러 개여도 도트는 1개 (책 단위 중복 제거)
- 도트 순서: `colorIndex` 오름차순

#### 위클리 / 먼슬리 공용
- 기존 `DayCell`을 확장하여 도트 배열을 받도록 수정
- 위클리: 도트 크기 6pt
- 먼슬리: 도트 크기 4pt (셀이 더 작으므로)

### 3. 달력 하단 — 책 표지 카드 목록

기존 `DaySessionList`(세션 시간 리스트)를 **책 표지 카드 목록**으로 교체한다.

#### 카드 레이아웃

```
┌──────────────────────────────────┐
│ 🔴 ┌──────┐  클린 코드            │
│    │ 표지 │  1시간 30분            │
│    │ 이미지│                      │
│    └──────┘                      │
├──────────────────────────────────┤
│ 🟠 ┌──────┐  함께 자라기           │
│    │ 표지 │  45분                  │
│    │ 이미지│                      │
│    └──────┘                      │
└──────────────────────────────────┘
```

#### 카드 구성 요소

| 위치 | 요소 | 설명 |
|------|------|------|
| 좌측 상단 | 컬러 뱃지 | 해당 책의 `BookColor` 원형 (10pt) |
| 좌측 | 책 표지 이미지 | `AsyncImage`로 `coverURL` 로드, 없으면 플레이스홀더 |
| 우측 상단 | 책 제목 | 1줄, lineLimit |
| 우측 하단 | 읽은 시간 | **해당 날짜**의 세션 duration 합계 (DurationFormatter 사용) |

#### 데이터 집계
- 선택된 날짜의 세션을 `libraryBookId` 기준으로 그룹핑
- 같은 책의 세션 duration을 합산
- `colorIndex` 오름차순 정렬

#### 빈 상태
- 세션 없는 날: "이 날은 독서를 하지 않았어요" (기존 동일)
