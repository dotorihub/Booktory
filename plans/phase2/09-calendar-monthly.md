# 09 — 먼슬리 달력

## 개요

기록 탭의 달력 섹션 중 먼슬리 뷰.
월 단위 그리드 달력으로 독서 활동을 한눈에 볼 수 있고, 날짜 탭 시 해당 날의 세션 상세를 확인한다.

**의존**: `08-calendar-weekly` (토글 구조 공유)

---

## 사용자 시나리오

1. 기록 탭에서 [먼슬리] 탭 전환
2. 이번 달 달력 표시
3. 독서한 날짜가 강조 표시됨
4. 날짜 셀 탭 → 해당 날 세션 목록 표시
5. 월 이동 (이전 달 / 다음 달)

---

## 기능 명세

### 달력 표시
- 현재 월 기준 그리드 (7열, n행)
- 해당 월의 1일 시작 요일에 맞게 오프셋 처리
- 이전 달 / 다음 달 이동 가능 (미래 달은 이동 불가)

### 날짜 셀 표현
- 위클리와 동일한 색 depth 로직 적용
- 오늘 날짜 강조
- 현재 월이 아닌 날짜 (앞/뒤 공백): 비어있거나 흐리게

### 날짜 탭 시
- 해당 날의 세션 리스트를 달력 하단에 표시
- 세션 없으면 빈 상태 메시지

### 월 이동
- 상단 "2025년 3월" 헤더 + < > 이동 버튼
- 미래 달로는 이동 불가 (오늘 기준)
