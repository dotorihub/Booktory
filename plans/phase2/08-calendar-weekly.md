# 08 — 위클리 달력

## 개요

기록 탭의 달력 섹션 중 위클리 뷰 (기본 뷰).
최근 7일 또는 이번 주를 가로로 나열하고, 각 날짜의 독서 여부 / 독서 시간을 시각적으로 표현한다.

**의존**: `07-stats`

---

## 사용자 시나리오

1. 기록 탭 진입 → 위클리 뷰가 기본 표시
2. 각 날짜 셀에서 그날의 독서 여부 확인
3. 날짜 셀 탭 → 해당 날의 세션 목록 상세 표시 (하단 시트 또는 인라인)
4. [먼슬리 보기] 전환 버튼 탭 → 09-calendar-monthly로 전환

---

## 기능 명세

### 날짜 표시 범위
- 이번 주 월~일 7일

### 날짜 셀 표현
- 독서 없는 날: 비어있음 (기본 원형 셀)
- 독서한 날: 색상 강조 (독서 시간에 비례하는 색 depth)
  - 30분 미만: 연한 강조
  - 30분~2시간: 중간 강조
  - 2시간 이상: 진한 강조
- 오늘 날짜: 테두리 또는 별도 표시

### 날짜 탭 시
- 해당 날의 세션 목록을 하단에 인라인으로 표시
- 세션이 없으면 "이 날은 독서를 하지 않았어요"

### 뷰 전환
- 위클리 / 먼슬리 토글 버튼 (달력 섹션 상단)

## 데이터

- `ReadingSession` 이번 주 날짜 범위 필터
- 날짜별로 그룹핑 (`Dictionary(grouping:by:)`)
- 날짜별 총 독서 시간 계산
