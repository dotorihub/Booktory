# 07 — 기록 탭 통계 수치

## 개요

탭 3(기록)의 상단 통계 섹션. 쌓인 `ReadingSession` 데이터를 집계하여
이번 주 독서 시간, 전체 누적 시간, 완독 권수를 수치로 보여준다.

**의존**: `00-swiftdata`, `04-timer-session` (세션 데이터가 쌓여 있어야 의미 있음)

---

## 사용자 시나리오

1. 기록 탭 진입
2. 상단에서 이번 주 / 전체 통계 확인
3. 완독한 책 수 확인

---

## 기능 명세

### 통계 항목

| 항목 | 계산 방법 |
|---|---|
| 이번 주 독서 시간 | 이번 주 월~일 날짜 범위의 `ReadingSession.duration` 합계 |
| 전체 누적 독서 시간 | 모든 `ReadingSession.duration` 합계 |
| 완독한 책 | `status == .completed`인 `LibraryBook` 수 |

### 시간 포맷
- 1시간 이상: "X시간 Y분"
- 1시간 미만: "Y분"
- 0분: "0분"

### 이번 주 기준
- 월요일 ~ 일요일 (Calendar.current 기준)
