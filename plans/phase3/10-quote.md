# 10 — 문장 / 이미지 기록

## 개요

독서 중 인상적인 문장이나 책 페이지 이미지를 기록하는 기능.
타이머 화면에서 진입하며, 텍스트 입력과 카메라/사진 촬영을 지원한다.

**의존**: `00-swiftdata`, `04-timer-session`

> **Phase 3 기능** — MVP 이후 구현 예정

---

## 사용자 시나리오

### 시나리오 A — 텍스트 기록
1. 타이머 화면에서 [문장 기록] 버튼 탭
2. 텍스트 입력 화면 오픈 (타이머는 유지)
3. 문장 입력 후 저장
4. 타이머 화면으로 복귀

### 시나리오 B — 이미지 기록
1. 타이머 화면에서 [카메라] 버튼 탭
2. 카메라 또는 사진 앨범 선택
3. 촬영/선택 후 저장
4. 타이머 화면으로 복귀

### 시나리오 C — 기록 보기
1. 기록 탭 → 문장 기록 섹션
2. 책별 / 날짜별 기록 목록 확인
3. 서재 책 상세 → 해당 책의 기록 목록 확인

---

## 기능 명세

### Quote 콘텐츠 타입
- `text`: 텍스트 입력 (멀티라인)
- `image`: 사진 촬영 또는 앨범에서 선택 (`Data`로 저장)

### 타이머 화면 진입 버튼
- 텍스트 입력 버튼 (키보드 아이콘)
- 카메라 버튼 (카메라 아이콘)
- 두 버튼 모두 타이머를 일시정지하지 않고 오버레이/시트로 표시

### 저장 데이터
```swift
Quote(
    id: UUID,
    libraryBookId: UUID,       // 현재 읽고 있는 책
    contentType: .text / .image,
    textContent: String?,
    imageData: Data?,
    createdAt: Date
)
```

│                             │
└─────────────────────────────┘
